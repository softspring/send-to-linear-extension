Support

For installation help, bug reports, or privacy questions, contact soporte@softspring.es.

Support policy

Send To Linear is a free Chrome extension that Softspring provides on an "As Is" basis. The source code is publicly available, and we may review reported issues when time allows, but we cannot guarantee responses, fixes, ongoing maintenance, or compatibility.

Recommended support details

When reporting an issue, include the following:

  • Your Chrome version.
  • The page where you tried to create the issue.
  • The team you selected.
  • Any error message shown by the extension.

Common setup checks

  • Confirm the extension is connected to Linear with a completed OAuth login.
  • Refresh teams after connecting or reconnecting the account.
  • Test on a normal `https://` page first, not a Chrome internal page.
